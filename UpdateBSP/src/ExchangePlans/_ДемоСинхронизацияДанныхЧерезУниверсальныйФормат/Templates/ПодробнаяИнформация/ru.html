Демо: Синхронизация данных через универсальный формат предназначена для демонстрации настройки и выполнения синхронизации данных между конфигурациями Демонстрационная конфигурация "Библиотека стандартных подсистем", редакция 3.0 через универсальный формат данных Enterprise Data. Синхронизация данных является демонстрационной и не предназначена для работы с реальными прикладными данными. Выполняется синхронизация следующих данных:

  • Справочник "Валюты"
  • Справочник "Демо: Банковские счета"
  • Справочник "Демо: Договоры контрагентов"
  • Справочник "Демо: Контрагенты"
  • Справочник "Демо: Места хранения"
  • Справочник "Демо: Номенклатура"
  • Справочник "Демо: Организации" - выполняется также синхронизация контактной информации.
  • Справочник "Демо: Физические лица"
  • Справочник "Классификатор банков"
  • Справочник "Пользователи"
  • Справочник "Страны мира" - элементы справочника синхронизируются по ссылке с дополнительным сопоставлением по коду и наименованию. Поскольку пользователи могут заполнять справочник произвольными странами (например, таможенными союзами для учета стран происхождения товара), которых нет в классификаторе стран мира (ОКСМ), то целевой объект сначала ищется по ссылке (если он был введен в одной из программ и был ранее синхронизирован), а затем, в случае неудачи, по составному ключу "Код" + "Наименование" (если выполняется синхронизация стран, введенных вручную или загруженных из классификатора независимо в обеих программах).
  • Регистр сведений "Демо: Ответственные лица"
  • Документ "Демо: Корректировка остатков товаров в местах хранения"
  • Документ "Демо: Поступление товаров"
  • Документ "Демо: Реализация товаров"

При настройке можно также указать организации, по которым будут синхронизироваться данные. Также при настройке можно задать дату, начиная с которой будут синхронизироваться заказы покупателей.


Для демонстрации синхронизации данных в обе стороны необходимо развернуть вторую информационную базу.
